## Features
- Add and remove selected books
- Search for books using Google books and Open Library API

## Tools Used ‚öôÔ∏è

- [React Native](https://reactnative.dev/) - Mobile Application
- [Expo](https://docs.expo.dev/) - Mobile Application
- [StepZen](https://stepzen.com/) - Build a GraphQL API, using public REST API's such as **Google Books API** and **Open Library API** 
- [Apollo Client](https://www.apollographql.com/docs/react/integrations/react-native/) (to query the API from React Native)
- [Async Storage](https://react-native-async-storage.github.io/async-storage/) (to persist data on the device storage)
- [Open Library API](https://openlibrary.org/developers/api)
- [Google Books API](https://developers.google.com/books)
- Styled Components
- CSS

## Future Features
